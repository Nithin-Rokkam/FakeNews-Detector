# Fake-News-Detector
Engineered an innovative API that seamlessly integrated real-time news fetching with
cutting-edge machine learning and NLP models; achieved a 95% accuracy rate in classifying
over 10,000 news articles as credible or fake.
The model takes real-time news based on News.org API key and predict its credibility.
The system classifies content as real or fake based on trained models.
Tech: Python, HTML, CSS, Java Script, NLP models, Machine Learning, Flask API
